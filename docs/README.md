# Präsentation für GitHub Pages

Dieser Ordner enthält die **Projektpräsentation** als eigenständige Website. Er wird von GitHub Pages genutzt, wenn unter **Settings → Pages** der Ordner **/docs** als Quelle eingestellt ist.

**Link zu den Folien:** [beex2008.github.io/data-analyst-onlineshop](https://beex2008.github.io/data-analyst-onlineshop/)

- `index.html` – Reveal.js-Präsentation
- `assets/` – Bilder und Grafiken für die Slides

Die eigentliche Quell-Präsentation liegt unter `presentation/` im Projektroot.
